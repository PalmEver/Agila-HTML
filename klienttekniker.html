Klienttekniker


Här kommer vi kortfattat gå igenom de olika delarna som används
för att bygga denna webbsida.


HTML


HyperText Markup Language (HTML) är själva skelettet för en webbsida, ett märkspråk som används för att bygga upp själva strukturen för webbsidan. HTML är uppbyggt av attribut och element som indikerar vad som ska visas på sidan. Elementet består av start- och sluttaggar där innehållet placeras, rubriker, paragrafer, listor m.m. Attributen beskriver källan till bilder som ska visas eller vilken adress länken länkar till.

JavaScript


JavaScript är musklerna för webbsidan, ett scriptspråk som används för att skapa bland annat funktioner och händelser. Det går att skriva JavaScript direkt i html dokumentet med hjälp av script taggar, men det rekommenderas att använda en separat fil. Då kan samma funktioner användas på olika sidor, genom att länka till den internt.

CSS


Cascading Style Sheets (CSS) är skinnet för webbsidan, ett språk bestående av regler som bestämmer hur elementen ska presenteras. I CSS anger man layouten, färger, bakgrundsbilder/färger, teckensnitt m.m. Man kan även använda CSS för att göra webbsidan responsiv samt anpassa den för olika mediatyper och skärmstorlekar.
