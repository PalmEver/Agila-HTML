Klient-server modellen


En klient-server modellen är ett nätverkssystem där servrar centralt hantera klienter eller arbetsstationer. Arbetsstationer är till exempel vanliga stationära och bärbara datorer, medan servrar oftast är mer kraftfulla datorer som utför särskilda funktioner. Servrar är dock inte själva datorn utan ett program som lika gärna kan vara installerat på en laptop, men för att klara av trafiken och av säkerhetsskäl är det ej rekommenderat. Klienten kan vara webbläsare som begär informationen (till exempel en webbsida) och servern levererar informationen.

Protokoll


Ett protokoll är regler som hjälper datorer att komunicera med varandra, så de talar samma språk. Men för att två datorer ska kunna komunicera med varandra behövs mer än ett protokoll. IP berättar hur datornerna ska prata med varandra och TCP sköter själva leveransen av informationen.

TCP står för Transmission Control Protocol och är ett av internets fundament och ser till att informationen som skickas över internet kommer fram. TCP bygger på något som kallas "packet switching" som innebär att paketet delas upp i mindre och numrerade paket, vart paketen sedan färdas och byggs ihop styrs av TCP. Anledningen till att det delas upp är för att göra komunikationen snabbare. Man kan jämnföra det en motorväg, om det blir för mycket trafik på motorvägen kan paketen ta en omväg runt motorvägen för att snabbare nå slutmålet.
Internetprotokollet (IP) standardiserar hur datorer komunicerar med varandra och likt ett telefonnummer eller en e-postadress får varje internetansluten dator en egen IP-adress.


http / https


http/https står för HyperText Transfer Protocol och används av webbsidor för att kommunicera mellan webbläsare och servrar.
s:et i https står för Secure och skillnaden mellan http och https är att https är krypterad kommunikation och därmed säkrare att använda om man hanterar känslig information.


ftp / ftps


ftp/ftps står för File Transfer Protocol och är ett kommandobaserat protokoll för överföring av filer, till exempel en webbsida.
Även här står s:et för Secure och har samma funktion som tidigare nämnt, informationen krypteras och gör överföringen säkrare.
